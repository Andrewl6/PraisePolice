# PraisePolice / 表揚警察
Scan下編號, 輕鬆認識你身邊嘅警察！

功能:
- 用相機Scan編號
- 人手輸入編號
- 離線資料庫

## Setup
1. Clone
2. Signup for firebase and obtain your own `google-service.json`
2. Put your own `google-service.json` inside `app/`
